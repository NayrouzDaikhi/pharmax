### Step 1: Download Ollama (2 minutes)
1. Go to: https://ollama.com/download
2. Click "Download for Windows"
3. Wait for `OllamaSetup.exe` to download

### Step 2: Install Ollama (1 minute)
1. Double-click `OllamaSetup.exe`
2. Follow the installation wizard
3. Click "Finish"
4. Ollama will start automatically

### Step 3: Download the AI Model (5-10 minutes)
Open Command Prompt or PowerShell and run:
```bash
ollama pull mistral
```
This downloads the AI model (~4GB). This only happens once.

### Step 4: Verify Ollama is Running
In PowerShell/Command Prompt, run:
```bash
curl http://localhost:11434/api/tags
```

Should return something like:
```json
{"models":[{"name":"mistral:latest",...}]}
```
‚úÖ If you see this, Ollama is working!

## üÜò Troubleshooting

### Problem: "ollama: command not found"
**Solution**: Ollama not installed. Download from https://ollama.com/download

### Problem: "Connection refused"
**Solution**: 
1. Ollama not running - start it:
   ```bash
   ollama serve
   ```
2. Keep the window open while using chatbot

### Problem: "Model not found"
**Solution**: Download model:
```bash
ollama pull mistral
```

### Problem: Slow responses
**Solution**: 
- Use lighter model: `ollama pull orca-mini`
- Or add GPU support (see OLLAMA_SETUP_GUIDE.md)
